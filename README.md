## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Set the `VITE_FIREBASE_VAPID_KEY` in [.env.local](.env.local) to your Firebase Web Push certificates key
4. Run the app:
   `npm run dev`
